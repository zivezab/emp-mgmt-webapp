# Emp-Mgmt-Webapp
MVP employee salary management webapp
## Quick Start
Step 1: Make sure [Docker](https://docs.docker.com/) installed on your machine
```
docker-compose up -d --build
```
Step 2: Open browser

[http://localhost:8000/](http://localhost:8000/)
